# 함께찾개 – 기능 정의서

## 1) 핵심 목적
유기견/유실견 매칭률 향상  
특히 **강아지 시절 사진만 존재하는 경우에도 → 성견 모습 예측 + 유사견 탐색 가능하도록**

## 2) 주요 기능 정의

| 기능명 | 설명 | 예시 Use case |
|---|---|---|
| 성견 이미지 생성 | 업로드된 강아지 사진 기반으로 성견 시기 외모 예상 생성 | 4개월짜리 강아지 사진 → 1~2년 성장한 이미지 생성 |
| 유사견 검색 | CLIP 임베딩 기반 유사도 계산 → DB 내 유기견 이미지 비교 | 생성된 성견 이미지로 보호소 등록 개체 중 유사한 개체 추천 |
| 필터 기반 검색 | 품종/색상 조건 기반 탐색 | 진돗개 + 화이트 계열만 필터 |
| 반려동물 신고 | 사용자가 발견한 강아지 사진 등록 → 추후 유사견 탐색 가능 | 산책중 길에서 발견한 강아지 신고 |
| 유기견 입양 정보 조회 | 보호소 등록 유기견 상세 정보 열람 | 보호소 DB 기반 정보 출력 (등록번호/품종/위치 등) |
| 마이페이지 | 신고 이력 / 검색 이력 / 즐겨찾기 관리 | 사용자가 관심 저장 후 나중에 다시 탐색 |

## 3) 기술 기반
- 이미지 생성: Stable Diffusion + LoRA + IP-Adapter
- 이미지 검색: CLIP 임베딩 기반
- 색/품종 정보 추출: YOLOv8 + Segmentation + K-means
