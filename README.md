# 함께찾개
AI 기반 성견 이미지 생성 및 유기견 이미지 검색 서비스  

---

## 1. 프로젝트 개요
- 현재 유기견 문제는 연간 10만 마리 이상 발생하며 반환율은 약 11% 수준으로 매우 낮음  
- 특히 **만 1세 미만 강아지는 성장 후 외모 변화 폭이 커** 기존 얼굴 기반 매칭 서비스로는 성공적인 매칭에 어려움이 존재  
- 본 프로젝트는 **강아지 시절 사진만 있어도 → 성견 모습 예측 + 유사견 탐색**을 가능하게 하여 반환 성공률을 높이는 것을 목표로 함

---

## 2. 핵심 기능
| 기능 | 설명 |
|---|---|
| 유사견 검색 | 보호소/DB 내 등록 개체와 비교하여 유사한 강아지를 리스트업 |
| 성견 이미지 생성 | 강아지 사진을 기반으로 성견 시기의 외모 과학적 예측 생성 |
| 필터 기반 검색 | 품종/색상 등 조건 기반 검색 지원 |

---

## 3. 아키텍처 개요
- 이미지 생성: Stable Diffusion + LoRA + IP-Adapter
- 이미지 검색: CLIP 기반 임베딩 + YOLOv8 품종/색 정보 추출
- 색 정보 매칭: CIELAB ΔE 활용
- 아키텍처 상세 이미지:
  - docs/architecture_generation.png  ← 첫 이미지 생성 흐름
  - docs/architecture_search.png      ← 두번째 검색 흐름

## 4. 사용 기술 스택
| 영역 | 기술 |
|---|---|
